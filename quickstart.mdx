---
title: "Get started with js-ini-parser"
description: "Learn how to install js-ini-parser, import parseIni and stringifyIni, and write your first INI parse and stringify operation in under 5 minutes."
---

This guide walks you through installing js-ini-parser and writing your first parse and stringify operation. By the end you'll have a working TypeScript snippet that reads an INI string, accesses section data, and converts it back to text.

<Steps>
  <Step title="Install the package">
    Add js-ini-parser to your project using your preferred package manager.

    <CodeGroup>
      ```bash npm
      npm install js-ini-parser
      ```
      ```bash yarn
      yarn add js-ini-parser
      ```
      ```bash pnpm
      pnpm add js-ini-parser
      ```
    </CodeGroup>
  </Step>

  <Step title="Import the functions">
    js-ini-parser ships both ESM and CommonJS builds.

    <Tabs>
      <Tab title="ESM (TypeScript / Node.js ESM)">
        ```typescript
        import { parseIni, stringifyIni } from "js-ini-parser";
        ```
      </Tab>
      <Tab title="CommonJS">
        ```javascript
        const { parseIni, stringifyIni } = require("js-ini-parser");
        ```
      </Tab>
    </Tabs>
  </Step>

  <Step title="Parse an INI string">
    Pass your INI content to `parseIni` along with a `ParserOptions` object.

    ```typescript
    import { parseIni } from "js-ini-parser";

    const ini = `
    [server]
    ; this is a comment
    host = localhost
    port = 8080
    `;

    const parsed = parseIni(ini, {
      allowGlobalSection: false,
      allowEmptyValue: false,
    });

    console.log(parsed);
    ```

    <Note>
      `parseIni` returns an `IniObject`, which is an array of `Section` objects. Each `Section` has a `section` name and a `blocks` array of `Block` entries (either data key-value pairs or comments). See the [Types reference](/api/types) for details.
    </Note>
  </Step>

  <Step title="Stringify back to INI text">
    Use `stringifyIni` to convert the parsed object back to an INI-formatted string.

    ```typescript
    import { parseIni, stringifyIni } from "js-ini-parser";

    const ini = `
    [server]
    ; this is a comment
    host = localhost
    port = 8080
    `;

    const parsed = parseIni(ini, {});
    const text = stringifyIni(parsed, {});

    console.log(text);
    // [server]
    // ; this is a comment
    // host=localhost
    // port=8080
    ```
  </Step>
</Steps>
