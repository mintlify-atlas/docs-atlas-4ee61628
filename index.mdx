js-ini-parser is a lightweight TypeScript library for reading and writing INI configuration files. It parses INI text into structured JavaScript objects, preserves inline comments, supports global sections, and converts objects back to INI format — all with zero dependencies. It works in both Node.js and the browser.

## Quick example

```typescript
import { parseIni, stringifyIni } from "js-ini-parser";

const ini = `
[server]
; server hostname
host = localhost
port = 8080
`;

const parsed = parseIni(ini, { allowEmptyValue: false });
// Returns structured Section[] with blocks for each key/comment

const text = stringifyIni(parsed, {});
// Converts back to INI-formatted string
```
